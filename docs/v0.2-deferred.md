# Deferred out of v0.2

Ideas that surfaced during v0.2 and were deliberately not built, per
CLAUDE.md's scope rule. Each has enough context to pick up cold.

## P3 cannot verify ldflags-injected versions (build-to-tempdir)

L1's P3 runs `version.command` via `go run ./cmd/foo --version`. `go run`
compiles from HEAD but does **not** apply the target Makefile's `-ldflags`,
so for a target whose release version is injected with
`-X .../version.Version=...`, P3 sees the *source default*, not the injected
release string.

Today this is masked: Columbo's own source default (`Version = "0.1.0-pre"`)
equals what its Makefile injects, so `go run ./cmd/columbo version` and the
ldflags-built binary both print `0.1.0-pre`. A target where the two differ
(any real release build) would make P3 either false-PASS or be forced to
SKIP. P2 (source-side const drift) still works; it's P3 — the
binary-vs-build drift detector, the literal F001 detector — that goes dark.

This matters most at the **v1.0 Columbo self-audit**: Columbo uses ldflags,
so the drift probe this whole repo family exists to never ship broken would
be blind on Columbo itself.

**Fix shape:** build the real binary with the target's build flags into a
temp dir (the build-to-tempdir option v0.2 declined to keep the schema
small), then run *that* binary for P3. Needs the schema to express "this
build command produces binary X" or a convention for locating the artifact.
Required before the self-audit; not before v0.3.

## Live serverInfo drift probe (the other half of seed L1)

The seed L1 (`seed/harness/unix-socket/lanes/L1-build-invariants.sh`) reads
the MCP `serverInfo` over the wire and checks it against the binary version.
v0.2 shipped L1-static only and left this out because it needs an MCP stdio
client. Port the handshake from `seed/harness/unix-socket/mcp_sock.py` (the
reference impl, including its adaptive-recv-timeout patch) into a shared
`internal/probes` client. That same client is the dependency for L2 and L6,
so it lands as part of v0.3, and the live drift probe rides along with it.
